# 🧠 Quiz App Challenge

Chào mừng bạn đến với thử thách phát triển ứng dụng **Quiz System**! Đây là một nền tảng thi trắc nghiệm trực tuyến giúp bạn rèn luyện kỹ năng phát triển tính năng Real-time và quản lý dữ liệu lớn.

## 🎯 Mục tiêu
Xây dựng hệ thống ngân hàng câu hỏi, tổ chức phòng thi trực tuyến và bảng xếp hạng thời gian thực.

## 🚀 Cách bắt đầu
1. **Fork/Clone repo này**.
2. **Kiểm tra Tab Issues**: Toàn bộ User Stories đã được tạo sẵn.
3. **Thiết lập GitHub Project** để quản lý tiến độ.
4. **Code & PR**: Tuân thủ quy trình trong `CONTRIBUTING.md`.

## 📋 Cấu trúc Repo
- `/tasks`: Backlog chi tiết.
- `.github`: Issue & PR templates.
- `CONTRIBUTING.md`: Quy trình làm việc nhóm.
